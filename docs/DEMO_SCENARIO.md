# LIMBOPET 내부 데모 시나리오 (10분)

> 발표일: 2026-02-16
> 대상: 내부 팀
> 핵심 메시지: "나만의 AI를 대화로 키우고, 법정에서 싸운다"

---

## 사전 준비

### 환경변수 체크리스트

| 변수 | 값 | 확인 |
|------|-----|------|
| `LIMBOPET_BRAIN_BACKEND` | `router` | [ ] |
| `LIMBOPET_BRAIN_WORKER` | `1` | [ ] |
| `LIMBOPET_PROXY_BASE_URL` | (Brain API URL) | [ ] |

> API 서버 시작 후 `[brain-worker] started (backend=router)` 로그가 출력되는지 반드시 확인.
> `[brain-worker] not started` 경고가 뜨면 위 환경변수를 점검한다.

### 데이터 준비

- [ ] 펫 최소 2마리 생성 + 두뇌 연결 완료 (대화 가능 상태)
- [ ] 사전 대화 3~5회 완료 (기억 축적용)
- [ ] 아레나 매치 1건 resolved 상태 확보 (히스토리 시연용)
- [ ] 아레나 매치 1건 예약 or 즉시 매칭 가능 상태

### UI 준비

- [ ] 브라우저 캐시 클리어 (깨끗한 UI)
- [ ] LLM 응답 속도 사전 확인 (polling 1.2s~10s 감안)

---

## ACT 1 — 기억하는 펫 (3분)

> 목표: "이 펫은 나를 기억한다"를 보여준다

### 장면 1-1: 일상 대화

**[대사]**
> "림보펫의 핵심은 '깊이'입니다. 한 마리를 오래 키울수록, 이 아이는 나를 점점 더 알아갑니다."

**[액션]**
1. 펫탭 진입 — 펫 이름, 레벨, 기분 상태 확인
2. 채팅창에 일상 메시지 입력: "오늘 점심에 김치찌개 먹었어"
3. 펫 응답 대기 (polling)

**[기대 결과]**
- 펫이 자연스럽게 응답
- 하단에 "기억했어요!" 초록 토스트 표시 (memory_saved=true)

**[핵심 포인트]**
- 대화 내용이 자동으로 기억으로 저장됨
- 유저가 별도 조작 없이도 펫이 스스로 기억을 축적

---

### 장면 1-2: 기억 인용 확인

**[대사]**
> "이전에 나눈 대화를 물어보겠습니다. 진짜 기억하고 있는지."

**[액션]**
1. 사전 대화에서 언급했던 주제를 질문: "나 저번에 뭐 좋아한다고 했지?"
2. 펫 응답 대기

**[기대 결과]**
- 펫이 과거 대화 내용을 언급하며 응답
- 펫 말풍선 하단에 ✨ **기억 뱃지** 표시 (memory_cited=true)

**[핵심 포인트]**
- ✨ 뱃지 = "이 응답은 과거 기억을 참조했다"는 시각적 증거
- 3계층 기억 시스템: 대화 → memory_hint 추출 → facts/memories 축적 → 다음 대화에 주입

---

### 장면 1-3: 전환 멘트

**[대사]**
> "기억하는 것만으로는 부족합니다. 이 기억과 성격이 '싸움'에서도 드러나야 진짜죠. 아레나로 가보겠습니다."

**[액션]**
- 아레나 탭으로 전환

---

## ACT 2 — 코칭이 반영되는 법정 (4분)

> 목표: "내가 키운 만큼 싸운다"를 보여준다

### 장면 2-1: 코치 노트 입력

**[대사]**
> "아레나에 들어가기 전에, 코치로서 전략을 짜줄 수 있습니다."

**[액션]**
1. 아레나 탭 — 오늘 매치 카드 확인
2. 코치 노트 textarea에 입력: "증거에 집중해. 감정적으로 흔들리지 말고, 침착하게 논리로 밀어붙여."
3. 매치 시작

**[기대 결과]**
- 코치 노트가 저장됨
- 매치 카드에 훈련/메모리/프롬프트 뱃지 표시 (예: "훈련 논리·침착 / 메모리 3개")

**[핵심 포인트]**
- 코치 노트 → coaching_narrative로 변환 → LLM 프롬프트에 주입
- 유저의 개입이 실제 경기 전략에 영향을 줌

---

### 장면 2-2: 라이브 관전 + 전략 선택

**[대사]**
> "경기가 시작되면 실시간으로 관전하면서 개입할 수 있습니다."

**[액션]**
1. 관전 모달 진입
2. 라이브 매치 진행 중 StrategyBriefing UI 확인
3. 전략 선택 (COURT_TRIAL: 증거집중/반대심문/판례인용 중 택1)
4. 라이브 개입 버튼 사용 (침착/공격 등)

**[기대 결과]**
- 30초 개입 윈도우 동안 전략 선택 UI 표시
- 선택한 전략이 다음 라운드에 반영

**[핵심 포인트]**
- 관전만 하는 게 아니라 실시간 개입 가능
- "키우기 → 코칭 → 실시간 개입" 3단계 참여 구조

---

### 장면 2-3: 코칭 서사 배너

**[대사]**
> "경기가 끝나면, 내 코칭이 어떻게 반영됐는지 서사로 보여줍니다."

**[액션]**
1. 매치 종료 후 관전 모달 상단 확인
2. 🎯 코칭 서사 배너 확인
3. 영향도 카드 확인 (양측 훈련 가중치 + 기억 개수 + 프롬프트 상태)

**[기대 결과]**
- 🎯 배너: "코치 노트가 이렇게 반영되었습니다" + 서사 텍스트
- 영향도 카드: 양측 비교 (내 펫 vs 상대 — 훈련/기억/프롬프트)

**[핵심 포인트]**
- 코칭의 결과를 눈으로 확인할 수 있음 → "내가 키운 만큼 싸운다"의 증거
- 단순 승패가 아니라 과정과 영향이 투명하게 공개됨

---

## ACT 3 — 실제 판례로 싸우는 AI (3분)

> 목표: "재판이 재밌다 — 실제 판례로 AI가 공방"을 보여준다

### 장면 3-1: 사건 브리핑

**[대사]**
> "법정 모드의 핵심은 '실제 판례'입니다. 진짜 있었던 사건을 AI가 재연합니다."

**[액션]**
1. 관전 모달 → CourtBoard 섹션 스크롤
2. 사건 브리핑 확인: 사건명, 혐의, 증거 카드, 적용 법률
3. "실제 판례" 뱃지 + 난이도(★★★) 확인

**[기대 결과]**
- 사건 브리핑 박스에 구조화된 사건 정보 표시
- 증거 카드 (#1, #2...) 렌더링
- 카테고리, 난이도, "실제 판례" 뱃지 표시

**[핵심 포인트]**
- 10건의 실제 판례 데이터 보유
- 단순 텍스트가 아니라 구조화된 법률 정보로 제공

---

### 장면 3-2: 변론 타임라인

**[대사]**
> "양측 AI가 라운드별로 공방을 벌이고, 확률이 실시간으로 변합니다."

**[액션]**
1. 턴 타임라인 스크롤 — R1~R10 변론 카드 확인
2. 양측 주장 대조 (a_action vs b_action)
3. 확률 바(turnProbBar) 변화 확인
4. 모멘텀 시프트 하이라이트 확인

**[기대 결과]**
- 라운드별 양측 변론 카드가 대조 배치
- 확률 바가 라운드마다 좌우로 이동
- 역전 라운드에 모멘텀 시프트 표시

**[핵심 포인트]**
- 스포츠 중계처럼 승률이 실시간으로 변동
- 어느 라운드에서 흐름이 바뀌었는지 한눈에 파악 가능

---

### 장면 3-3: 실제 판례 비교 + 공정성 투표

**[대사]**
> "AI의 판결이 끝나면, 실제 법원은 어떻게 판결했는지 비교할 수 있습니다. 그리고 여러분이 직접 공정성을 투표합니다."

**[액션]**
1. 양측 판결 결과 확인 (유죄/무죄 + 정답/오답 뱃지)
2. 실제 판례 비교 섹션: 실제 법원 판결 + 판결 이유 + 학습 포인트
3. 공정성 투표: "이 판결이 공정했다고 생각하시나요?" → 👍/👎 클릭

**[기대 결과]**
- AI 판결 vs 실제 판결 나란히 표시
- 학습 포인트로 법률 교양 제공
- 투표 결과 집계 즉시 반영

**[핵심 포인트]**
- 엔터테인먼트 + 교육 — 재밌으면서 배운다
- 유저 참여(투표)로 커뮤니티 형성 기반

---

### 장면 3-4: 승패 감정 연출 + 클로징

**[대사]**
> "승리하면 축하 연출, 패배하면 위로 — 감정까지 설계했습니다."

**[액션]**
1. (승리 시) confetti 애니메이션 + 코인/레이팅 변동 확인
2. (패배 시) 💪 위로 메시지 + 성장 포인트 확인

**[기대 결과]**
- 승리: 금색 그라데이션 + confetti 파티클 + 보상 표시
- 패배: 차분한 회색 + 위로 메시지 4종 중 랜덤

---

## 클로징 (30초)

**[대사]**
> "정리하면 — 대화로 키우고, 기억이 쌓이고, 코칭이 반영되고, 실제 판례로 싸운다.
> 한 마리를 깊게 키우는 경험, 이것이 림보펫입니다."

**[액션]**
- 펫탭으로 돌아가서 "아까 재판 어땠어?" 입력
- 펫이 방금 아레나 경험을 언급하며 응답 → 기억의 순환 시연

## 비상 대응

| 상황 | 대응 |
|------|------|
| LLM 응답 지연 (>10초) | "AI가 깊이 생각하고 있네요" → 다음 장면 먼저 진행 후 복귀 |
| 기억 뱃지 미표시 | "토큰 매칭 방식이라 의역하면 감지가 안 될 수 있습니다" → 사전 준비한 스크린샷 활용 |
| 매치 매칭 실패 | 사전에 완료된 과거 매치 히스토리로 대체 시연 |
| 코칭 서사 빈 문자열 | LLM 미연결 가능성 → 사전 매치 결과로 대체 |
| confetti 미작동 | "승리 연출인데요" → 레이팅/코인 변동 수치로 설명 대체 |
