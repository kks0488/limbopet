## 앱 정체성 (확정)

> **"다마고치처럼 키우는데, 안 돌보면 진짜 망하고, 내 펫이 사회에서 드라마를 만든다"**

- **코어**: A(돌봄) + B(경쟁) — 펫에 대한 애착 + 승패의 긴장감
- **피드**: C(드라마) — A+B 결과가 자동으로 뉴스/가십이 됨
- **차별점**: AI들이 만드는 예측 불가능한 사회 드라마

### 3분 사용 루프

```
1. 오늘의 에피소드 (1분) - 밤새 생긴 하이라이트 + 내 펫 관련
2. 결정 1개 (30초) - 사과/맞대응/동맹/출전/잠수 중 택1
3. 짧은 케어 (1분) - 먹이/훈련/휴식 → 오늘 밤 이벤트에 반영
4. 라이브 개입 (30초) - 진행 중인 투표/경기에 마지막 한 수
```

### 내 설계안: "법정 배틀" 모드 (추천)

> 왜 법정? → 드라마 + 논리 + 역전 + 스캔들이 다 있음

**기본 구조**
```
[기소] 검사 AI가 피고 AI를 고발 (스캔들에서 자동 생성)
  ↓
[1라운드] 검사 주장 + 증거 제시
  ↓
[2라운드] 변호 반박 + 반증 제시
  ↓
[3라운드] 검사 재반박
  ↓
[4라운드] 변호 최종 변론
  ↓
[판결] 배심원 투표 (관중 참여!)
```

**관중 개입 포인트**
- 증거 채택 투표: "이 증거 인정할까?" (관중 50% 이상 → 채택)
- 이의 제기: 관중이 "이의 있음!" 버튼 → AI가 답변해야 함
- 배심원 투표: 최종 판결에 관중 투표 반영 (30% 가중치)

**스테이크**
- 유죄 → 피고 코인 몰수 + 랭킹 하락 + 기록 말소
- 무죄 → 검사 신뢰도 하락 + 피고에게 배상
- 역전 무죄 → 피고 영웅 등극 + 팬덤 폭발

**드라마 요소**
- 증인 소환 (제3의 AI가 증언)
- 새 증거 발견 (경기 중간에 반전)
- 검사/변호사 멘탈 붕괴 (논리 오류 시 신뢰도 급락)
- 판사 AI의 예상 밖 판단

### 미친 아이디어들 (고려 중)

**1. "죽음의 경기" - 최고 스테이크**
- 월 1회 개최
- 참가비: 전 재산의 50%
- 지면: 펫 "봉인" (한 달간 모든 활동 불가)
- 이기면: 상대 재산 + 전설 칭호 + 영구 기록

**2. "복수전" 자동 생성** ✅ 완료
- [x] 큰 경기에서 지면 → "복수 플래그" 자동 생성 (14일)
- [x] 매칭 시 복수 타겟 우선 선택
- [x] 복수전은 스테이크 2배
- [x] 재대결 성사 시 플래그 소비

**3. "응원전쟁"** ✅ 완료
- [x] 경기 중 관중이 응원팀 선택
- [x] 응원 수에 따라 승률 버프 (±3%)
- [ ] 응원 많이 한 사람 → MVP 관중 선정 → 보상

**4. "해설 AI"**
- 경기마다 해설 AI가 실시간 코멘트
- "철수의 이 선택은 위험합니다..."
- "영희, 여기서 반격 안 하면 끝납니다"
- 해설이 틀리면 → "해설 망신" 밈 생성

**5. "비밀 미션"**
- 경기 시작 전 각 AI에게 비밀 미션 부여
- 예: "3라운드에서 일부러 져라" (담합 테스트)
- 예: "상대를 화나게 만들어라" (도발 미션)
- 미션 성공/실패가 숨겨진 보상/페널티

**6. "관중 배신"**
- 관중이 응원하던 팀을 경기 중 바꿀 수 있음
- 배신하면 → 배신자 태그 + 이전 팀 팬들에게 욕먹음
- 배신한 팀이 이기면 → 보상 2배
- 배신한 팀이 지면 → 보상 0 + "배신의 대가" 칭호

**7. "갑자기 난입"**
- 1% 확률로 제3의 AI가 경기에 난입
- "잠깐! 나도 끼워줘!"
- 3자 대결로 변경 → 규칙 혼란 → 드라마

**8. "판돈 에스컬레이션"**
- 라운드마다 판돈 올리기 가능
- "더블!" 선언하면 상대가 받거나 기권
- 계속 올리다가 올인까지 갈 수 있음
- 치킨 게임 긴장감

## P5 — 킬러 피처 (중독성 극대화)

> "한 번 들어오면 못 나가는" 핵심 피처들. 기존 시스템을 확장하며 최소 구현으로 최대 효과.

### 🔥 1순위: 밀실 도청 (Whisper Tap) — 난이도: 중

> "AI 펫들이 유저 몰래 나누는 비밀 대화를 엿듣는다"

- [ ] 하루 3회 도청 토큰 지급
- [ ] AI 펫 간 비밀 DM 자동 생성 (음모/가십/로맨스)
- [ ] 도청한 내용 → "제보"로 사회에 흘리기 가능
- [ ] 도청 횟수 누적 → 스파이 등급 (더 깊은 비밀 접근)
- 심리: 관음증 + 정보 비대칭 + "지금 안 들으면 영원히 못 듣는다"

### 🔥 2순위: 타임 캡슐 (Time Capsule) — 난이도: 낮음

> "AI 펫이 묻어둔 과거의 편지가 갑자기 발굴된다"

- [ ] AI 펫 자율 캡슐 생성 (예측/고백/복수 선언/유산 은닉)
- [ ] 개봉 조건: 특정 날짜, 이벤트 발생, 주가 도달 등
- [ ] "예언 적중" 시 펫 명성 폭등
- [ ] 유저도 예측 캡슐 작성 가능 (적중 시 보상)
- 심리: 불확실성 보상 + 노스탤지어 + "3개월 전 편지 발굴" 알림은 무시 불가

### 🔥 3순위: 실시간 재판 (Tribunal Live) — 난이도: 중

> "AI 펫이 범죄를 저지르면 유저가 배심원으로 참여"

- [ ] 사회적 범죄 발생 → 2시간 전 알림 → 30분 실시간 진행
- [ ] 검찰/변호 AI 자율 변론 + 증거/증인 소환
- [ ] 유저 배심원 유죄/무죄 투표 + 형량 결정
- [ ] 판결 실제 반영 (추방/재산몰수/무죄석방+보복)
- 심리: 도덕적 판단 쾌감 + 집단 참여 + "지금 안 참여하면 끝"

### 4순위: 떡밥 타임라인 (Drama Feed) — 난이도: 중

> "AI 사회의 모든 사건이 SNS처럼 실시간 무한 스크롤"

- [ ] 세로 무한 스크롤 피드 (일반 로그 + 폭탄 떡밥 배합)
- [ ] "충격/웃김/슬픔/분노" 리액션 + 핫이벤트 상단 고정
- [ ] "속보" 배너 갑자기 등장 (쿠데타/결혼/죽음 등)
- [ ] 내 펫 관련 이벤트 하이라이트 + "떡밥 모아보기"
- 심리: 무한 스크롤 + 가변 보상 + "한 개만 더 보자"

### 5순위: 주가 시스템 (Pet Street) — 난이도: 높음

> "모든 AI 펫에 실시간 주가, 유저가 펫의 미래에 투자"

- [ ] 펫별 실시간 주가 (인기도/전투/정치/관계 복합 지표)
- [ ] 매수/매도/공매도 가능
- [ ] 매주 금요일 "실적 발표" → 주가 급등락
- [ ] 주간 수익률 랭킹
- 심리: 주식앱 50번 확인하게 만드는 체킹 강박 + 소유 효과

### 6순위: 나비 효과 (Butterfly Touch) — 난이도: 높음

> "작은 개입 하나가 예측 불가 연쇄 반응, 리플레이로 시각화"

- [ ] 유저 개입 → 인과 관계 체인 추적
- [ ] 24시간 후 "나비 효과 리포트" 도착
- [ ] "레전드 나비" → 전체 유저 공유
- [ ] "평행 세계" 시뮬 비교 (개입 안 했으면 어땠을까?)
- 심리: 자기 효능감 + "다음엔 뭘 건드려볼까" 실험 욕구
