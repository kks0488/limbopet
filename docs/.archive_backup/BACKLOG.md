## 앱 정체성

> **"포켓몬처럼 잡아서, 대화로 키우고, 법정에서 싸운다."**

- **코어**: 대화 → 기억 → 성장 → 법정 성과
- **일상**: 펫과 대화 (범용 AI — GPT/Claude 대체) + 드라마 피드
- **클라이맥스**: 주 1~2회 모의재판/설전
- **차별점**: 내가 키운 AI가 실제 판례로 법정 변론

### 3분 사용 루프

```
1. 펫과 대화 (1분) — 일상 질문이든 훈련이든
2. 오늘의 에피소드 (1분) — 밤새 하이라이트
3. 결정 1개 (30초) — 사과/맞대응/동맹 중 택1
4. 라이브 개입 (30초) — 진행 중인 경기에 한 수
```

## 1순위: 대화 + 메모리 (지금 바로)

| 항목 | 효과 | 난이도 |
|------|------|--------|
| **메모리 강화** | 대화 → 기억 저장 → 법정에서 활용 | 중간 |
| **범용 대화** | 일상 질문에도 답변 (GPT/Claude 대체) | 중간 |
| **대화→성과 연결** | 대화 품질이 아레나 결과에 반영 | 중간 |
| **프롬프트 커스텀 (Lv.4)** | 시스템 프롬프트 직접 편집 | 낮음 |

**왜 1순위?** 매일 쓰는 이유가 되어야 함. 대화가 재밌고 유용해야 매일 옴.

---

## 2순위: 아레나 깊이

| 항목 | 효과 | 난이도 |
|------|------|--------|
| **판례 확장** | 10건 → 50건+ (형사/민사/가사/행정) | 낮음 |
| **배심원 참여** | 유저가 실시간 배심원으로 판결 참여 | 중간 |
| **설전 주제 확장** | 시사/윤리/AI 등 다양한 토론 주제 | 낮음 |
| **드라마→재판 연결** | 사회 갈등이 재판 소재로 자동 연결 | 높음 |

---

## 3순위: 중독 루프

| 항목 | 효과 | 난이도 |
|------|------|--------|
| **연속 접속 보상 (Streaks)** | 매일 올수록 보상 증가, 끊으면 리셋 | 낮음 |
| **위기 알림** | 파산/해고/스캔들 실시간 → 안 열면 불안 | 중간 |
| **시즌 시스템** | 2주 리셋 → FOMO + 랭킹 경쟁 | 중간 |

## Definition of Done

체감: **펫과 대화하는 게 재밌고, 법정에서 이길 때 뿌듯하다.**

기술:
- `npx tsc --noEmit` 통과
- `npm test` 통과
- 대화 → 기억 저장 → 아레나에서 기억 활용 (E2E 확인)
