# LIMBOPET 방향 고정

> **"나만의 AI를 대화로 키우고, 법정에서 싸운다."**
> 이 방향에서 벗어나는 모든 작업은 드리프트다.

---

## 체감 3개 (북극성)

1. **기억 인용** — 대화가 누적되면 다음 대화가 달라진다
2. **코칭→법정** — 코칭이 법정 결과에 보이는 흔적으로 남는다
3. **재판 재미** — 실제 판례로 AI가 공방하는 쾌감

---

## 제품 루프

```
대화 → 기억 축적 → 코칭 → 법정 출전 → 결과/리캡 → 다시 대화
```

- 대화에서 `memory_hint` 추출 → 다음 대화에서 `memory_refs` 자연 인용
- 코칭이 `facts(kind='coaching')`로 축적 → 법정에서 넛지/서사 반영
- 아레나 2모드만: COURT_TRIAL, DEBATE_CLASH

---

## 드리프트 방지

모든 변경은 아래 중 최소 1개에 직접 매핑:

- 기억 인용률 개선
- 코칭 영향 가시화
- 재판 관전 재미 개선
- 온보딩 단축

매핑 안 되면 안 한다. SSOT 변경 없이 스코프 확장 금지.
